# E-Commerce RESTful API

A clean, scalable, and deployed backend API for an E-Commerce platform built using **Node.js, Express.js, MongoDB**.

**Live Server:** [https://send-production-06cf.up.railway.app](https://send-production-06cf.up.railway.app)

---


## Key Features

* **Auth & User Profiles:** JWT & bcrypt authentication, User/Admin roles, and shipping address management.
* **Product Catalog & Search:** Full CRUD for products, categories, subcategories, brands, plus search, filtering, sorting, pagination.
* **Cart, Orders & Payments:** Shopping cart, discount coupons, order processing, and online payments via **Stripe**.
* **Wishlist & Reviews:** Personal wishlist tracking and product reviews with dynamic ratings.
* **Media & Uploads:** Multi-image uploads and static file serving for products, brands, and categories.



## Tech Stack & Architecture

* **Runtime & Framework:** Node.js, Express.js
* **Database:** MongoDB (Mongoose ODM)
* **Auth & Security:** JWT, bcrypt, validation
* **Integrations & Uploads:** Stripe API (Payments), Multer 
* **Architecture:** Modular MVC structure, Factory functions pattern, and centralized error handling.
